# Installation

Get Ventyd installed and choose your preferred validation library.

## Install Ventyd

Install the core Ventyd package using your favorite package manager:

```bash tab="pnpm"
pnpm add ventyd
```

```bash tab="yarn"
yarn add ventyd
```

```bash tab="npm"
npm install ventyd
```

## Choose a Validation Library

Ventyd uses the [Standard Schema](https://standardschema.dev) specification, which means you can use any compatible validation library. We provide official integrations for the most popular options.

### Valibot (Recommended)

Valibot is lightweight, modular, and has excellent TypeScript support.

```bash tab="pnpm"
pnpm add valibot
```

```bash tab="yarn"
yarn add valibot
```

```bash tab="npm"
npm install valibot
```

**Why Valibot?**
- Smallest bundle size (~1KB for basic schemas)
- Modular - only import what you need
- Excellent TypeScript inference
- Fast validation performance

### TypeBox

TypeBox provides JSON Schema validation with strong typing.

```bash tab="pnpm"
pnpm add @sinclair/typebox @sinclair/typemap
```

```bash tab="yarn"
yarn add @sinclair/typebox @sinclair/typemap
```

```bash tab="npm"
npm install @sinclair/typebox @sinclair/typemap
```


**Why TypeBox?**
- JSON Schema compatible
- Runtime type checking
- OpenAPI integration
- Excellent for API validation

### Zod

Zod is popular, well-established, and has a large ecosystem.

```bash tab="pnpm"
pnpm add zod
```

```bash tab="yarn"
yarn add zod
```

```bash tab="npm"
npm install zod
```


**Why Zod?**
- Large community and ecosystem
- Comprehensive documentation
- Many existing integrations
- Fluent, chainable API

### ArkType

ArkType offers runtime type safety with a unique syntax.

```bash tab="pnpm"
pnpm add arktype
```

```bash tab="yarn"
yarn add arktype
```

```bash tab="npm"
npm install arktype
```


**Why ArkType?**
- Ultra-fast validation (up to 10x faster than Zod)
- Elegant, concise syntax
- Excellent error messages
- Great TypeScript integration

### Other Libraries

Any library that implements the [Standard Schema](https://standardschema.dev) specification will work with Ventyd. You can also use the `standard()` provider directly.

## Verify Installation

Create a simple test file to verify your installation:

```typescript title="test.ts"
import { defineSchema, defineReducer, Entity } from 'ventyd';
import { valibot, v } from 'ventyd/valibot'; // or your chosen library

const testSchema = defineSchema("test", {
  schema: valibot({
    event: {
      created: v.object({
        message: v.string()
      })
    },
    state: v.object({
      message: v.string()
    })
  }),
  initialEventName: "test:created"
});

const testReducer = defineReducer(testSchema, (prevState, event) => {
  if (event.eventName === "test:created") {
    return { message: event.body.message };
  }
  return prevState;
});

class Test extends Entity(testSchema, testReducer) {}

const test = Test.create({
  body: { message: "Hello, Ventyd!" }
});

console.log("✅ Ventyd is installed correctly!");
console.log("Message:", test.state.message);
```

Run it:

```bash
npx tsx test.ts
```

If you see the success message, you're all set!
