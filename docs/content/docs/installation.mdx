## Install Ventyd

```bash tab="pnpm"
pnpm add ventyd
```

```bash tab="yarn"
yarn add ventyd
```

```bash tab="npm"
npm install ventyd
```

<Callout type="warn">
Ventyd requires TypeScript 5.0 or higher with `strict: true`
</Callout>

## Choose a Validation Library
