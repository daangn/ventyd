# Ventyd Skills

Agent skill for the [ventyd](https://github.com/daangn/ventyd) TypeScript event sourcing library.

Ventyd Skills are folders of instructions and resources that coding agents can discover and use to gain ventyd knowledge. They contain setup instructions, best practices, and methods for retrieving current documentation.

## Installation

Install using any coding agent that supports the [Skills standard](https://agentskills.io):

```bash
npx skills add daangn/ventyd
```

```bash
pnpm dlx skills add daangn/ventyd
```

```bash
yarn dlx skills add daangn/ventyd
```

```bash
bun x skills add daangn/ventyd
```

## Compatibility

Works with any coding agent that supports the Skills standard, including:

- [Claude Code](https://docs.anthropic.com/en/docs/claude-code)
- [Cursor](https://cursor.com)
- [Codex](https://openai.com/codex)
- [OpenCode](https://opencode.ai)

Also available on [GitHub](https://github.com/daangn/ventyd).

## What's Included

- **SKILL.md** - Core ventyd knowledge: architecture, patterns, critical rules, and complete examples
- **references/quick-start.md** - Project setup and first entity guide
- **references/embedded-docs.md** - How to find API documentation in installed packages
- **references/remote-docs.md** - How to access ventyd.com documentation
- **references/common-errors.md** - Troubleshooting guide for frequent errors
